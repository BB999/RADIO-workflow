# kamuicode Workflows

🎨 Claude Code SDK と kamuicode MCP を活用したAIコンテンツ生成ワークフロー集

## 🌟 概要

GitHub Actionsを使用してAIコンテンツを自動生成する包括的なワークフローコレクション。プロフェッショナルなラジオ番組からニュース動画、バナー広告から音楽動画まで、数クリックで高品質なメディアコンテンツを作成できます。

**最新追加**: 🎙️ **ラジオワークフロー** - 開発レポートを4分間のプロ品質ラジオ番組に変換する完全自動化システム。AI音声、BGM、ユーザー投稿楽曲を統合。

## 📦 ワークフロー一覧

### 🎙️ ラジオワークフロー 🆕
**完全自動ラジオ番組制作システム**

任意のテキストコンテンツをプロフェッショナルなラジオ番組に変換。AI音声合成、BGM生成、ユーザー楽曲統合機能付き。

**主要機能:**
- 🎯 **240秒** プロ品質ラジオ番組生成
- 🎤 **AIVIS Cloud API** - 高品質20代女性AI音声
- 🎵 **Google Lyria API** - 自動BGM生成  
- 📻 **ユーザー楽曲統合** - 投稿楽曲からランダム選択・紹介
- 📺 **動画出力** - MP3音声 + 静止画をMP4動画化
- 🔊 **放送品質** - -23 LUFS準拠音声レベル
- ⚡ **5セクション構成** - オープニング、メイン（前後半）、ジングル、エンディング

**最適用途:**
- 開発進捗レポート
- 製品発表
- コミュニティ更新情報
- 教育コンテンツ
- ブランドストーリーテリング

### 🏗️ モジュールワークフロー
**モジュラー・マルチエージェント・コンテンツ生成**

様々なコンテンツタイプに対応する再利用可能コンポーネントのスケーラブルシステム。

**機能:**
- 📰 プロフェッショナルニュース動画制作
- 🎬 マルチモデル動画生成（画像5種類 + 動画3種類）
- 🖼️ バナー広告作成（最大4バリエーション）
- 🎯 テキスト→動画、画像→動画、参照→動画
- 📝 Gemini Vision解析による自動字幕
- 🌍 自動翻訳対応多言語サポート

### 🎵 音楽動画ワークフロー
**音楽 + 動画統合**

Google Lyria音楽生成とImagen4 + Hailuo-02 Pro動画生成を組み合わせた音楽動画作成。

### 🎨 画像ワークフロー
**マルチシーン画像生成**

単一プロンプトからImagen4 Ultraを使用して3つの異なるシチュエーション画像を生成。

### 📹 動画ワークフローテンプレート
**基本動画生成**

学習用途や基本的な使用例向けのシンプルで初心者にやさしい動画作成ワークフロー。

### 🎬 背景除去付き動画ワークフロー
**高度動画処理**

背景除去機能を内蔵した動画生成ワークフロー。

### 🔍 Gemini I2V解析ワークフロー
**解析付き画像→動画変換**

解析機能付き画像から動画生成のためのGemini API統合ワークフロー。

## 🚀 クイックスタート

1. **上記リストからワークフローを選択**
2. **ワークフローフォルダに移動**してセットアップガイドを読む
3. **必要なシークレット設定**とMCP設定を行う
4. **GitHub Actions経由でワークフローを実行**

## 🛠️ 技術要件

### コアインフラ
- **GitHub Actions** - ワークフロー実行環境
- **Claude Code SDK** - AIエージェントランタイム
- **kamuicode MCP** - マルチモデルコンテンツ生成サービス
- **Anthropic API Key** - Claude AIアクセス

### 専用API
- **🎙️ AIVIS Cloud API** - プロフェッショナルAI音声合成（ラジオワークフロー用）
- **🎵 Google Lyria API** - AI音楽生成（ラジオ/音楽動画用）
- **🧠 Gemini API Key** - 映像解析・字幕生成（推奨）
- **🔧 FFmpeg** - プロフェッショナル動画編集（ニュース/ラジオ動画出力用）

### GitHub統合
- **GitHub PAT Token** - 自動プルリクエスト作成
- **LINE通知** - 完了アラート（オプション）

## 🎙️ ラジオワークフローの特色

### 音声リソース管理
- **ユーザー音楽ライブラリ**: `radio-workflow/music/` - 自動楽曲選択・紹介
- **ジングル効果音**: プロフェッショナル転換音
- **マルチ音声サポート**: ランダムAI音声パーソナリティ選択

### 制作品質
- **放送基準**: -23 LUFS音声レベル準拠
- **高品位**: 44.1kHzステレオ出力
- **プロ構成**: 5セクションラジオ形式
- **時間ベースランダム化**: 音楽・音声選択のバリエーション確保

### ワークフロー知能
- **モジュラー処理**: 効率化のための並列生成  
- **エラーハンドリング**: 自動リトライ（最大3回）
- **品質検証**: 音声ファイル検証
- **包括ログ**: 詳細な制作統計

## 🌟 kamuicode Workflowsを選ぶ理由

- **🔄 完全自動化** - コンセプトから完成品まで
- **🎯 制作レディ** - プロフェッショナル品質出力
- **🛠️ モジュラー設計** - コンポーネントの組み合わせ可能
- **📈 スケーラブル** - 単体アイテムからバッチ処理まで対応
- **🚀 使いやすさ** - GitHub Actionsインターフェース
- **🔧 カスタマイズ可能** - プロンプト・パラメータ変更可能
- **📊 透明性** - 詳細ログ・統計情報
